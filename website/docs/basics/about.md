---
title: What is RadiantScript?
sidebar_label: About RadiantScript
---

RadiantScript is a high-level programming language for smart contracts on Radiant Blockchain. It offers a strong abstraction layer over Radiant's native virtual machine, enabling developers to write powerful smart contracts with features unique to Radiant including:

- **Reference-based tokens** - Native fungible and non-fungible tokens without overlay protocols
- **Transaction introspection** - Access to full transaction data within contracts
- **State management** - Persistent contract state via the state separator mechanism
- **Conservation rules** - Enforce token supply constraints at the consensus level

RadiantScript's syntax is based on Solidity, making it familiar to Ethereum developers while providing access to Radiant's unique UTXO-based capabilities.

If you're interested to see what kind of things can be built with RadiantScript, check out the [Examples](/docs/language/examples) or the standard library templates in the repository. To get started, refer to the [Getting Started page](/docs/basics/getting-started).
