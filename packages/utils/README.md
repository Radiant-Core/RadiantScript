# @radiantscript/utils

[![NPM Version](https://img.shields.io/npm/v/@radiantscript/utils.svg)](https://www.npmjs.com/package/@radiantscript/utils)
[![NPM License](https://img.shields.io/npm/l/@radiantscript/utils.svg)](https://www.npmjs.com/package/@radiantscript/utils)

`@radiantscript/utils` is a package containing utilities and types used by both the RadiantScript compiler (`rxdc`) and the JavaScript SDK (`radiantscript`). This package can be used by other applications as well, but it should be considered an unstable API. If you want to use `@radiantscript/utils` as a standalone package, it is recommended to specify a specific version in your `package.json`.
